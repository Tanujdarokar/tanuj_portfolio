## ✨ Portfolio Redesign & Visual Features

This portfolio website has been fully redesigned into a state-of-the-art interactive experience:
- **Modern Dark-Mode Canvas:** A deep slate-black background featuring smooth, glowing backdrop blurs.
- **Glassmorphic UI Elements:** Navigation bars, card components, and input forms are styled as translucent sheets with fine borders and backdrop filter blurs.
- **Animated Background Blobs:** Floating neon gradient circular shapes that float in the background and interactive parallax shifting tied to mouse movements.
- **Typewriter Title Subheader:** A custom script loops through key stack profiles in the hero landing text with typewriter keyframe blinking selectors.
- **Micro-interactions & Speed:** GPU-accelerated transition translations (60fps) and asynchronously handled viewport link highlighting using native `IntersectionObserver`.
